Baidu Netdisk Description:

1、获取授权码，授权码可多次使用，建议复制保存便于下次使用。

1. Obtain the authorization code. The code can be used multiple times and it is recommended to copy and save for future use.

a. 手动授权（手动复制授权码，无法自动授权时，使用手动授权方式，授权成功后，返回链接携带 access_token=******，复制 ****** 令牌到授权码）

a. Manual authorization ( Copy the authorization code manually. When automatic authorization is not possible, use manual authorization. After successful authorization, the return link will carry the access_token=******, copy the ****** token into the authorization code field.)

2、不开通百度会员会受到下载限速，上传不会限速

2. Without Baidu membership, download speeds are limited, but upload speeds are not restricted.

3、支持秒传

3. Supports rapid upload.

4、关于分片上传大小文件限制

4. Regarding the file size limitations for chunked uploads

a. 普通用户单个分片大小固定为4MB，单文件总大小上限为4G

a. For regular users, the size of each chunk is fixed at 4MB, with a maximum file size of 4GB.

b. 普通会员用户单个分片大小上限为16MB，单文件总大小上限为10G

b. For regular member users, the maximum size of each chunk is 16MB, with a maximum file size of 10GB.

c. 超级会员用户单个分片大小上限为32MB，单文件总大小上限为20G

c. For super member users, the maximum size of each chunk is 32MB, with a maximum file size of 20GB.
